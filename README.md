# VUE Simple Quiz

Simple game with questions and 2 mini-games. Share result on Vk and Facebook.


## Example
[LInk](http://062019.kdeit.ru/)

## Use
PHP + VUE JS

## Installation



```bash
npm i
npm run build
or
npm run dev
```

## Usage

```javascript
Fill riddles on screen#.vue
Replace share images (./static/share)
http://localhost:8080/

```
